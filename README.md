# 기존 redux를 활용한 mbti프로젝트를 redux-toolkit을 활용해 수정해보기
~~~
npm install @reduxjs/toolkit
~~~

## 기존 사용스택 및 라이브러리(React)
~~~
npx create-react-app mbti-test

npm install react-redux redux
npm install styled-components

npm start
~~~
